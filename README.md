# ARS — 3D FPS Game

3D шутер от первого лица в браузере. Three.js + JavaScript.

## Как играть

Открой `index.html` в браузере (Chrome, Firefox, Edge).

Или играй онлайн: https://unidel2035.github.io/ars/

## Управление

- **WASD** — движение
- **Мышь** — обзор
- **ЛКМ** — стрельба
- **R** — перезарядка
- **Esc** — пауза

## Уровни

1. **Training Room** — маленькая комната, 3 врага
2. **Dark Corridors** — средняя комната с колоннами, 5 врагов
3. **The Arena** — большая арена с ящиками, 7 врагов

Убей всех врагов → появится зелёный портал → переход на следующий уровень.
